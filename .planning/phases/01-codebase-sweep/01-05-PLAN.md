Current pattern in src/DeepCWorld.cpp — BEFORE (buggy):

(see CONVENTIONS.md). Use `_inverse_window_matrix`.

case and no return after the closing brace (around lines 110-126). Add `default: return 0;`

returns const char*) as the last case in each respective switch.

Match the type of each function's return:
- `default_input()` returns `Op*` — add `default: return nullptr;` (C++17 style, consistent

- `input_label()` returns `const char*` — add `default: return "";`
