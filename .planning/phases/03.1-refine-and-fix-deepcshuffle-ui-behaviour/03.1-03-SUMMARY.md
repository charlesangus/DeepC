## Accomplishments

## Decisions Made
- `ChannelButton` uses direct `QPainter` rendering in `paintEvent` with no Qt stylesheet. This gives precise per-pixel control over the colored fill, border, and X-mark and avoids stylesheet cascading issues inside Nuke's panel.
- `const:0` button gets near-black (30, 30, 30) and `const:1` gets near-white (220, 220, 220) to visually communicate constant 0.0 and 1.0 source values respectively.
- The down-arrow row is a separate grid row (`arrowRow = startRow + 2`) which shifts `dataStartRow` from `+2` to `+3`. This is a clean approach that keeps headers and arrows independent rows.
- The right-arrow for output row labels is prepended as a UTF-8 string prefix inside the existing `outLabel` — no extra grid column needed.
