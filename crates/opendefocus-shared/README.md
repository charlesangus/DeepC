# OpenDefocus Shared

Crate that contains the source code that is shared between the [kernel](https://crates.io/crates/opendefocus-kernel) and the [OpenDefocus](https://crates.io/crates/opendefocus) core crate.

This is a subcrate of the [OpenDefocus](https://crates.io/crates/opendefocus) crate and is not meant to be used directly.
