foreach (z, requestedChannels)
{
  Channel inChannel = z;  // default: passthrough
  for (int i = 0; i < 8; ++i) {
    if (_outputChannels[i] != Chan_Black && z == _outputChannels[i]) {
      inChannel = _sourceChannels[i];

  outPixel.getWritableUnorderedSample(sampleNo, z) = inData;
}

**Class declaration (in-file; no separate .h needed for Op):**
Remove `_inChannel0..3` and `_outChannel0..3`. Add:
- `ChannelSet _in1ChannelSet` (default: `Mask_RGBA`)
- `ChannelSet _in2ChannelSet` (default: `Chan_Black`)
- `std::string _matrixState`
- `std::array<Channel, 8> _outputChannels` (default-init to `Chan_Black`)
- `std::array<Channel, 8> _sourceChannels` (default-init to `Chan_Black`)
- `int _activeSlots` (count of active routing slots, populated in _validate)

**Constructor:** Initialize `_in1ChannelSet = Mask_RGBA; _in2ChannelSet = Chan_Black; _activeSlots = 0;` Fill both arrays with `Chan_Black`.

**_validate():**
1. Call `DeepFilterOp::_validate(for_real)`
2. Reset: fill both arrays with `Chan_Black`, set `_activeSlots = 0`
3. If `_matrixState` is empty, update DeepInfo with just the input channels and return
4. Parse `_matrixState`: use `std::istringstream` to split by `,`; for each token split by `:` into `outName` and `srcName`; call `DD::Image::getChannel(outName.c_str())` and `DD::Image::getChannel(srcName.c_str())`; store in arrays up to slot 8; increment `_activeSlots`
5. Build `newChannelSet`: start from `_deepInfo.channels()`; foreach `_outputChannels[i]` where it != `Chan_Black`: add it to `newChannelSet`
6. Set `_deepInfo = DeepInfo(_deepInfo.formats(), _deepInfo.box(), newChannelSet)`

**getDeepRequests():**
Build `neededChannels` from `requestedChannels` plus all `_sourceChannels[i]` where != `Chan_Black`. Push `RequestData(input0(), bbox, neededChannels, count)`.

**doDeepEngine():**
Guard `if (!input0()) return true`. Build `neededChannels` same as getDeepRequests. Call `input0()->deepEngine(...)`. Loop pixels and samples. Inside `foreach(z, requestedChannels)`: initialize `Channel inChannel = z` then loop routing slots to check for override (see interface block above). Write output sample. Close loops. Assert complete, set `deepOutPlane`.

**knobs():** Exactly as shown in the interface block above.

**knob_changed():** Exactly as shown in the interface block above (push ChannelSets + updateWidgets).

**node_help():** Update text as shown in interface block above.

**Plugin registration:** Keep unchanged: `static Op* build(Node* node) { return new DeepCShuffle(node); }` and `const Op::Description DeepCShuffle::d("DeepCShuffle", 0, build);`
