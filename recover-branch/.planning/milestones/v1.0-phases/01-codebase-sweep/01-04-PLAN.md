<objective>
Remove DeepCBlink from the codebase entirely: delete the source file and excise both
its CMake build references.

Purpose: DeepCBlink ships broken behavior — the "Use GPU if available" knob never
activates GPU (hardcoded to CPU), silent bail-out on >4 channels, and two calloc
memory leaks with no matching free on early-exit paths. Removing it is cleaner than
keeping a misleading, incomplete node in the toolbar.

Output: DeepCBlink.cpp deleted, CMakeLists.txt cleaned of both the PLUGINS list entry
and the standalone target_link_libraries line. Human checkpoint confirms the build
succeeds and the node is absent from the toolbar.
</objective>
